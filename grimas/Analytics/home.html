Web Analytics: Network and SCF reports... without Alma!

Online Analytics Reports

The following links use the Analytics API to retrieve reports from Alma's Oracle Analytics Server and - in many cases - will pass user-supplied filters and prompts to refine the information. If you have an idea for a new report, please reach out to WRLC.

    Prompted Reports
  • [Retention Reassignment]-- Display all copies of a title in the network
  • [Journal Volume Overlap - Barcode]-- Display all volumes of a journal title already in the SCF (by Barcode)
  • [Journal Volume Overlap - ISSN]-- Display all volumes of a journal title already in the SCF (by ISSN)
  • [Tray Check]-- Display all barcodes currently in a shelf or tray at the SCF
